IntegraGuard Surveillance System

IntegraGuard is an advanced security system designed to **detect weapons like guns and knives** using **YOLOv8 object detection**. This project integrates a robust **backend for AI processing** and a **user-friendly frontend interface** for real-time detection results.

The system is built with **Python for AI/ML processing**, **OpenCV for image handling**, and a **lightweight web stack** for UI interaction.
